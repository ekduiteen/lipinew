================================================================================
TROUBLESHOOTING
================================================================================

SSH tunnel lost?
  → Run Step 1 again in Terminal 1

Backend can't reach database?
  → psql -h localhost -p 5432 -U lipi -d lipi (test connection)
  → Verify SSH tunnel is running with -L 5432 flag

Frontend can't reach backend?
  → curl http://localhost:8000/health (check backend is running)
  → Verify NEXT_PUBLIC_API_URL="http://localhost:8000"

Port already in use?
  → lsof -i :8000 (find process)
  → kill -9 <PID> (kill it)

vLLM slow to start?
  → This is normal, model is loading (~5-10 minutes first time)
  → curl http://localhost:8080/v1/models to check status
